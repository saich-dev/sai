Display Block:

It will take full width of screen and always start with new line.

Examples of Display Block
  • h1 to h6
  • Divition tag
  • Paragraph tag
  • Unorderlist tag
  • Orderlist tag
  • Forms tag
  • pre tag
  • menu tag
  • section tag
  • article tag
  • aside tag


Display inline:

It will take necessary space based on content and it starts with same line and it not allows to set a width and height on the element.

Examples of Display inline
  • Anchor tag
  • Strong tag
  • em tag
  • bold tag
  • quoatation tag
  • mark tag
  • span tag

Display inline-Block:

It will allows to set a width and height on the element and it allows top and bottom margins/paddings are respectively.

Examples of Display inline-Block
  • Button tag
  • input tag
  • Select tag
  • Textarea tag

Display None:

Any element can have its display set to none.
