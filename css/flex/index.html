What is Flex.
  • flex is an container which will have items and it is used for design layout.
  • we should avoid using table ,as table is used for showing data not to layout page.

What is container.
  • Elements used to define layout of its children.

What is Flex item.
  • All children of flex container is flex-item.
  • The layout of an item or element depends on the rule applied on container.
