What is Overflow

In Cascading Style Sheets (CSS), overflow occurs when an element's content doesn't fit inside the element's box


Overflow:Auto

This value is similar to scroll, but scrollbars are only added when the content actually overflows. If the content fits within the container, no scrollbars appear.
